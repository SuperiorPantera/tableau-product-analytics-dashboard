# Tableau Product Analytics Dashboard

## Overview
Developed interactive Tableau dashboards to analyze player engagement and revenue metrics for gaming and subscription-based products.

## Key Analyses
- Battle Pass engagement analysis
- Average playtime by age groups
- Activity heatmaps
- New MRR calculation
- Month-over-month Revenue growth
- Revenue cohort analysis

## Technologies
- Tableau
- LOD Expressions
- Table Calculations
